# 🐍 贪吃蛇游戏 (Snake Game)

一个适合手机端操作的经典贪吃蛇游戏，使用 HTML5 Canvas 和 JavaScript 开发。

## ✨ 特性

- 📱 **移动端友好** - 响应式设计，适配各种屏幕尺寸
- 🎮 **多种控制方式** - 虚拟方向键、键盘、滑动手势
- 🏆 **最高分记录** - 本地存储最高分
- 🎨 **精美界面** - 渐变色彩、动画效果
- ⚡ **动态加速** - 吃得越多，速度越快

## 🚀 快速开始

### 在线游玩
直接在浏览器中打开 `index.html` 文件即可开始游戏！

### 本地运行
```bash
# 使用 Python 启动简单服务器
python3 -m http.server 8080

# 或者使用 Node.js
npx serve .
```

然后访问 `http://localhost:8080`

## 🎯 游戏说明

### 控制方式

**手机端：**
- 点击屏幕上的方向按钮
- 或在游戏区域滑动手势

**电脑端：**
- 方向键 `↑↓←→`
- 或 `WASD` 键
- 空格键暂停/继续
- 回车键开始游戏

### 游戏规则
- 控制蛇吃掉红色食物
- 每吃一个食物得 10 分
- 蛇会随着分数增加而加速
- 撞到墙壁或自己游戏结束
